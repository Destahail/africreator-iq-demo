# AfriCreator IQ Demo

Investor-ready React demo for an East Africa Creator Intelligence Platform.

## What is included
- Homepage
- Creator Discovery
- Creator Detail
- Trends Intelligence
- AI Match Studio
- Agency Dashboard
- Saved Lists
- Admin Dashboard
- Add Creator
- Sync Center
- Pricing

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Upload this folder to GitHub.
2. Go to https://vercel.com/new
3. Import the GitHub repository.
4. Framework: Vite
5. Build command: `npm run build`
6. Output directory: `dist`
7. Deploy.
