# AfriCreator IQ Demo

Investor-ready Vite/React demo.

## Local run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Vercel settings
- Framework: Vite
- Build command: npm run build
- Output directory: dist

This demo uses local sample data. Future backend can connect Supabase and official TikTok sync.
